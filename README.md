# Smart Traffic Control System using VANET

## Project Overview
This project presents a prototype smart traffic monitoring and driver safety system using Vehicular Ad-hoc Networks (VANET) and IoT technologies.

The system detects nearby vehicles using ultrasonic sensors and provides safety alerts to drivers.

This project was developed as an undergraduate thesis in Computer Science and Engineering.

## Technologies Used
- Arduino UNO
- Ultrasonic Sensors
- Bluetooth Module
- Android Interface
- Firebase Database
- Java
- XML

## Features
- Vehicle distance detection
- Collision warning system
- Driver safety alert
- Traffic density monitoring
- Prototype VANET communication concept

## Hardware Components
- Arduino UNO
- Ultrasonic Sensors (3 units)
- Breadboard
- Bluetooth Module
- Power Supply (5V)
